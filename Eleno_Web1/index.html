The padding Property

Padding is the space between the border of a box and any content contained within it.

(Adding padding can increase the readability of its contents.)

Value

The value of this property is most often specified in pixels (although it is also possible to use percentages or ems). If a percentage is used, the padding is a percentage of the browser window (or of the containing box if it is inside another box).

You can specify different values for each side of a box using:padding-top padding-right padding-bottom padding-left

Or you can use a shorthand (where the values are in clockwise order: top, right, bottom, left):padding: 10px 5px 3px 1px;

Disclaimer

The value of the padding property is not inherited by child elements in the same way that the color value of the font-family property is; so you need to specify the padding for every element that needs to use it!

Fixing Width Conflict

There's one last thing you need to know about CSS padding, and itâ€™s the relationship padding has with CSS width property. CSS width describes how wide the content area (text, image, etc.) of an element is. The problem occurs when you determine width and padding separately. In such case, CSS adds padding value to the total width instead of considering padding a part of an element.

You can easily fix this issue with the box-sizing property. It defines how the width and height of an element are calculated: should they include padding and borders, or not. By using this property, you instruct the browser to decrease the possible content space instead of merely adding the padding to the total value.
